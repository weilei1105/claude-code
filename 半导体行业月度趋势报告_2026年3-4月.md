# 半导体行业月度趋势报告

**时间范围：2026年3月10日 - 4月10日**

---

## 一、市场总览

| 指标 | 数据 |
|------|------|
| 全球半导体销售额（2025年11月）| **$753亿**（同比+29.8%）|
| 2026年预计全年总规模 | **超过$1万亿** |
| 2026年晶圆代工产量增长预测 | **24.8%** |

## 三、设备与材料国产化

### 3.1 国产设备突破

**中安半导体**发布最新一代灵敏度颗粒缺陷检测设备 **ZP8**：

- 特征灵敏度达到 **10.5nm**
- 支持更先进制程的良率监控
- 从26nm到10.5nm的跨越，体现国产检测设备的快速迭代

### 3.2 掩模版与材料

| 进展 | 详情 |
|------|------|
| 掩模版项目签约 | 沪士电子AI芯片配套高端印制线路板项目签约，**总投资101亿元** |
| 光子器件 | 昭明半导体光子器件项目通线，**总投资26.5亿元**，年产1亿只 |
| 二维半导体材料 | WSi2N4作为P型沟道材料，在二维半导体CMOS集成电路中展现广阔前景 |
